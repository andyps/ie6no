@version 1.0

1. *********************************************************
Назначение этого скрипта - вывод предупреждающего окошка для пользователей 
устаревших версий браузера IE, а именно версий ниже седьмой.

Простой пример использования: 
<script type="text/javascript">ie6no({runonload:true});</script>
Предупреждающий элемент div будет добавлен в начало тела документа
автоматически после загрузки документа.

Пример ручного запуска:
ie6no().run();

После вызова метода run можно в последующем скрывать и снова отображать окошко,
например:
var warn = ie6no(); //инициализация
warn.run(); //создание и отображение окна 
warn.hide(); //скрытие окна
warn.show(); //отображение окна

При этом, если запуск идет автоматически после загрузки документа,
как в первом примере, вызывать run не нужно.

2. *********************************************************
Параметры, которые можно передавать в текущей версии:

target: 
        id элемента или существующий узел документа, куда будет добавляться
        шаблон html для ie6no;
runonload (true или false):
          автоматический запуск после загрузки страницы, если равен true;
isOverlay (true или false):
          использовать или нет фон-затемнение;
overlayBgColor:
                цвет затемнения, например #eeeeee;
overlayOpacity: 
                прозрачность затемнения от 0(полностью прозрачный фон) 
                до 100(полностью непрозрачный фон);
isCloseBtn (true или false):
                             показывать или нет кнопку закрытия, если такая есть;
closeBtnClass:
               имя класса для идентификации кнопки закрытия, если 
               используется собственный шаблон для окна;
               будет произведен поиск кнопки по данному имени класса 
               в содержимом шаблона, поэтому она должна там находиться;
zIndex:
        базовое числовое значение z-index; указанное значение будет 
        использоваться для слоя затемнения, для окошка будет использовано
        на 1 единицу большее значение;
        этот параметр может понадобиться менять, если окно или фон ie6no
        не перекрывают какие-либо элементы на странице, но нужно, чтобы 
        перекрывали;
tpl: 
     это html-шаблон для окошка; таким образом можно указать свой 
     html-код(используйте внимательно).
