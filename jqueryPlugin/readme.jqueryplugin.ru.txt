Как использовать jQuery плагин

Использование плагина аналогично использованию независимой версии. ie6no jQuery плагин не поддерживает версии IE ниже
шестой.

Таким образом, отличие только в том, что вместо ie6no 
нужно использовать $.ie6no. Например:
<script type="text/javascript">$.ie6no({runonload:true});</script>

Требуется, как минимум, jQuery 1.2.2.
